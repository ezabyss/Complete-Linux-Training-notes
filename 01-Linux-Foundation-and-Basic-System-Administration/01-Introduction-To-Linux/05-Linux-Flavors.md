# üêß Most Used Linux Flavors (Distributions)

When people talk about **Linux flavors**, they usually mean **Linux distributions (distros)**.

A **Linux distribution** is a complete operating system built using:
- The **Linux kernel**
- System utilities
- Package manager
- Desktop environment (optional)
- Configuration tools

Different distributions exist to serve **different users, goals, and environments**.
