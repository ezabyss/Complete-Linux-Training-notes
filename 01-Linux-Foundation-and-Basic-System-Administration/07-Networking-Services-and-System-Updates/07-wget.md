# 📥 Linux File Downloading (`wget`)
### Internet Download 🌐 | No GUI Needed 💻 | SysAdmin Essential 🧠

---

## 🎯 Why Do We Need `wget`?

In Windows:
- Open browser 🌐  
- Click download ⬇️  

But in Linux (especially servers):
- ❌ No GUI  
- ❌ No browser  

👉 So we need a **command-line way to download files**

---

## 🧠 Core Idea

> “Download files directly from the internet using terminal.”

---

## 🌍 General Understanding (Not Just Cybersecurity)

Downloading files = Getting something from the internet

Examples:
- App 📦  
- File 📄  
- Software 💻  
- Script ⚙️  

---

## 📌 What is `wget`?

`wget` = **Web Get**

👉 Meaning:
- Get files from the **World Wide Web**

---

## ⚙️ Basic Syntax

```bash
wget <URL>
```

---

## 🧪 Example

```bash
wget https://example.com/file.zip
```

---

## 📂 Where File Gets Saved?

```bash
pwd
```

---

## 🔍 Check Downloaded File

```bash
ls
```

---

## ❓ Why Not Use `yum install`?

| Tool | Use Case |
|------|--------|
| `yum install` | Official packages |
| `wget` | Any file from internet |

---

## 🌐 Real Scenario

- `yum` → ❌ not found  
- `wget` → ✅ works
